|-------------------+--------------------------------------------------------------------------------------|

| capture_wave      | Utility to assemble large waveforms for ADC characterisation                         |
| scripts           | General scripts and                                                                  |
| makeIocs          | Diamond iocbuilder sources                                                           |
| xml               | Curated Ethercat Slave Information files                                             |
|                   | These come mostly from ftp://ftp.beckhoff.com/Config/ethercat/XML_Device_Description |
| makeDocumentation | Doxygen documentation files                                                          |
| builder           | iocbuilder support                                                                   |
| debug             | Python mini master                                                                   |
| simulations       | Unknown                                                                              |
